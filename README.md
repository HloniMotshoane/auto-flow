# AutoFlow SA

A workflow automation platform tailored for the South African market, built to streamline business processes and enhance operational efficiency.

## Tech Stack

This project is built with a modern, type-safe stack:

- **Framework:** React + Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **UI Components:** shadcn/ui
- **Backend/Database:** Supabase (PostgreSQL)
- **State Management:** React Query (TanStack Query)

## Getting Started

To run the project locally, ensure you have Node.js & npm installed.

```bash
# 1. Clone the repository
git clone [https://github.com/HloniMotshoane/auto-flow.git](https://github.com/HloniMotshoane/auto-flow.git)

# 2. Navigate to the project directory
cd auto-flow

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
